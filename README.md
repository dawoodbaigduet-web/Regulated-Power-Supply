# Regulated Power Supply

A regulated DC power supply designed to provide stable and reliable output voltage for electronic circuits and embedded systems.

## Features
- Stable DC output voltage
- Low ripple output
- Voltage regulation
- Reliable power source for electronics projects

## Components
- Transformer
- Bridge Rectifier
- Filter Capacitor
- Voltage Regulator IC (7805/7812)
- Resistors
- Capacitors
- PCB

## Skills Used
- Electronics
- Circuit Analysis
- Power Electronics
- Soldering
- Testing & Troubleshooting

## Description
Designed, assembled, and tested a regulated DC power supply capable of converting AC input into a stable DC output. The project involved circuit design, component selection, soldering, voltage testing, and performance verification.

## Applications
- Arduino Projects
- Embedded Systems
- Robotics
- Electronic Circuit Testing
